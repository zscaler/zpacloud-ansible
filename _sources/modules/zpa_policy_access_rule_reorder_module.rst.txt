.. Title

zscaler.zpacloud.zpa_policy_access_rule_reorder module -- Triggers the reorder of all policy types.
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.0.6).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_policy_access_rule_reorder_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_policy_access_rule_reorder`.
