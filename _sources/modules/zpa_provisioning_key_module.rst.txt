.. Title

zscaler.zpacloud.zpa_provisioning_key module -- Create a Provisioning Key.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.0.0).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_provisioning_key_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_provisioning_key`.

Synopsis
--------

.. Description

- This module will create/update/delete a specific Provisioning Key by association type (CONNECTOR\_GRP or SERVICE\_EDGE\_GRP).

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get ID Information of a Connector Enrollment Certificate
      zscaler.zpacloud.zpa_enrollement_certificate_facts:
        provider: "{{ zpa_cloud }}"
        name: "Connector"
      register: enrollment_cert_connector

    - name: Get ID Information of a App Connector Group
      zscaler.zpacloud.zpa_app_connector_groups_facts:
        provider: "{{ zpa_cloud }}"
        name: "Example"
      register: app_connector_group

    - name: "Create/Update/Delete App Connector Group Provisioning Key"
      zscaler.zpacloud.zpa_provisioning_key:
        provider: "{{ zpa_cloud }}"
        name: "App Connector Group Provisioning Key"
        association_type: "connector"
        max_usage: "10"
        enrollment_cert_id: "{{ enrollment_cert_connector.data[0].id }}"
        zcomponent_id: "{{ enrollment_cert_connector.data[0].id }}"
