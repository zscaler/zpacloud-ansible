.. Title

zscaler.zpacloud.zpa_isolation_profile_info module -- Retrieves Isolation Profile information.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.4.5).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_isolation_profile_info_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_isolation_profile_info`.

New in zscaler.zpacloud 1.0.0

Synopsis
--------

.. Description

- This module will allow the retrieval of information about an Cloud Browser Isolation Profile from the ZPA Cloud.

------------
The below requirements are needed on the host that executes this module.

- Zscaler SDK Python can be obtained from PyPI \ `https://pypi.org/project/zscaler-sdk-python/ <https://pypi.org/project/zscaler-sdk-python/>`__

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get Details of All Cloud Browser Isolation profiles
      zscaler.zpacloud.zpa_isolation_profile_facts:
        provider: "{{ zpa_cloud }}"

    - name: Get Details of a Specific Cloud Browser Isolation profile by Name
      zscaler.zpacloud.zpa_isolation_profile_facts:
        provider: "{{ zpa_cloud }}"
        name: ZPA_CBI_Profile

    - name: Get Details of a specific Cloud Browser Isolation profile by ID
      zscaler.zpacloud.zpa_isolation_profile_facts:
        provider: "{{ zpa_cloud }}"
        id: "216196257331282583"
