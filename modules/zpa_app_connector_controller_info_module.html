zscaler.zpacloud.zpa_app_connector_controller_info module – Retrieves an app connector controller information

Note

This module is part of the zscaler.zpacloud collection (version 2.0.6).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it, use: ansible-galaxy collection install zscaler.zpacloud. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: zscaler.zpacloud.zpa_app_connector_controller_info.

New in zscaler.zpacloud 1.0.0

  • Synopsis

  • Requirements

  • Parameters

  • Notes

  • Examples

Synopsis

  • This module will allow the retrieval of information about an app connector controller.

Requirements

The below requirements are needed on the host that executes this module.

  • Zscaler SDK Python can be obtained from PyPI https://pypi.org/project/zscaler-sdk-python/

Examples

- name: Retrieve All App Connector Controllers
  zscaler.zpacloud.zpa_app_connector_controller_info:
    provider: "{{ zpa_cloud }}"

- name: Retrieve App Connector Controller By Name
  zscaler.zpacloud.zpa_app_connector_controller_info:
    provider: "{{ zpa_cloud }}"
    name: 'SJC037_App_Connector'

- name: Retrieve App Connector Controllers By ID
  zscaler.zpacloud.zpa_app_connector_controller_info:
    provider: "{{ zpa_cloud }}"
