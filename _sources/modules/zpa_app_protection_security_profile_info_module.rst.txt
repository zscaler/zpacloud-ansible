.. Title

zscaler.zpacloud.zpa_app_protection_security_profile_info module -- Retrieves App Protection Security Profile information.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.4.2).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_app_protection_security_profile_info_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_app_protection_security_profile_info`.

Synopsis
--------

.. Description

- This module will allow the retrieval of information about an App Protection Profile from the ZPA Cloud.

------------
The below requirements are needed on the host that executes this module.

- Zscaler SDK Python can be obtained from PyPI \ `https://pypi.org/project/zscaler-sdk-python/ <https://pypi.org/project/zscaler-sdk-python/>`__

Examples

    - name: Get Details of All App Protection profiles
      zscaler.zpacloud.zpa_app_protection_security_profile_facts:
        provider: "{{ zpa_cloud }}"

    - name: Get Details of a Specific App Protection profiles by Name
      zscaler.zpacloud.zpa_app_protection_security_profile_facts:
        provider: "{{ zpa_cloud }}"
        name: Example

      zscaler.zpacloud.zpa_app_protection_security_profile_facts:
        provider: "{{ zpa_cloud }}"
