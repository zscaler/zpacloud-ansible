zscaler.zpacloud.zpa_application_server_info module -- Retrieve an application server information.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.4.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_application_server_info_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_application_server_info`.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Gather Information Details of All Application Servers
      zscaler.zpacloud.zpa_application_server_facts:
        provider: "{{ zpa_cloud }}"

    - name: Gather Information Details of an Application Server by Name
      zscaler.zpacloud.zpa_application_server_facts:
        provider: "{{ zpa_cloud }}"
        name: server1.acme.com

    - name: Gather Information Details of an Application Server by ID
      zscaler.zpacloud.zpa_application_server_facts:
        provider: "{{ zpa_cloud }}"
        id: "216196257331291921"
