zscaler.zpacloud.zpa_lss_config_log_types_formats_info module -- Retrieves LSS Log formats Information.
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.4.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_lss_config_log_types_formats_info_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_lss_config_log_types_formats_info`.

Synopsis
--------

.. Description

- This module will allow the retrieval of LSS (Log Streaming Services) Log formats information from the ZPA Cloud.

Requirements
------------
The below requirements are needed on the host that executes this module.

- Zscaler SDK Python can be obtained from PyPI \ `https://pypi.org/project/zscaler-sdk-python/ <https://pypi.org/project/zscaler-sdk-python/>`__
