.. Title

zscaler.zpacloud.zpa_saml_attribute_info module -- Retrieves saml attributes from a given IDP
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `zscaler.zpacloud collection <https://galaxy.ansible.com/ui/repo/published/zscaler/zpacloud/>`_ (version 1.4.2).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install zscaler.zpacloud`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.zscaler.zpacloud.zpa_saml_attribute_info_module_requirements>` for details.

    To use it in a playbook, specify: :code:`zscaler.zpacloud.zpa_saml_attribute_info`.

Synopsis
--------

.. Description

- This module will allow the retrieval of information about a saml attributes from a given IDP

------------

- Zscaler SDK Python can be obtained from PyPI \ `https://pypi.org/project/zscaler-sdk-python/ <https://pypi.org/project/zscaler-sdk-python/>`__

Examples
--------

    - name: Get Information About All SAML Attributes
      zscaler.zpacloud.zpa_saml_attribute_facts:
        provider: "{{ zpa_cloud }}"

    - name: Get Information About Saml Attribute by Attribute Name
      zscaler.zpacloud.zpa_saml_attribute_facts:
        provider: "{{ zpa_cloud }}"
        name: DepartmentName_User

    - name: Get Information About Saml Attribute by Attribute ID
      zscaler.zpacloud.zpa_saml_attribute_facts:
        provider: "{{ zpa_cloud }}"
